O que é o Painel Gerencial de Operações?

Aqui é onde as informações referentes às operações devem ser cadastradas

Qual a utilidade?

Para a polícia, o sistema servirá para facilitar a comunicação e notificação de operações ao MPRJ, bem como visualizá-las em lista para gerir atividades. Para o MPRJ, o sistema serve para tomar ciência das operações. Em última análise, o sistema auxilia no cumprimento de diversas obrigações legais e decorrentes de decisões judiciais.

Como funciona?

É um sistema de interface simples em que a polícia poderá cadastrar operações e visualizá-las em lista. Uma operação cadastrada poderá ter os dados atualizados ou corrigidos em momento posterior ao cadastro. Para isso, basta acessar a lista de operações e utilizar o formulário para correção.

Transparência nos dados?

Por ser uma plataforma compartilhada, as partes que a utilizam sabem o que todos os usuários veem, promovendo a equidade e transparência com relação às informações de operações fornecidas pela polícia.

Repositório de dados?

Para além da transparência, o sistema se destina a ser um repositório de dados de operação, permitindo a construção de inteligência sobre os fatos. Desde uma perspectiva de ciência de dados, é possível a elaboração de análises qualificadas no futuro.
